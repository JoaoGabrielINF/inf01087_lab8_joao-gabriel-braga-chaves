# Lab 8 - Uso Consciente de LLMs e Prompt Engineering

## Descrição da Atividade

A atividade consiste em elaborar prompts simples e detalhados, enviá-los a uma LLM, coletar suas respostas e analisá-las em diferentes contextos. O relatório abrange respostas conceituais sobre Alan Turing, geração de código para cálculo de Fibonacci, melhoria de prompts, verificação das respostas da IA e reflexão sobre o uso responsável de LLMs.

## AI Usage Statement

Este trabalho utilizou uma ferramenta de IA generativa para auxiliar na geração de explicações e exemplos de código.
Todos os prompts utilizados estão documentados em prompts.txt.
As respostas foram revisadas, testadas e verificadas pelo autor antes de serem incluídas no relatório.
